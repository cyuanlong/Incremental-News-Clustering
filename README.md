A source code for my thesis

## Instalation

Requirements

* Python 3.5
* Pip
* Pipenv

### Ubuntu

```bash
$ sudo apt-get install python3 python3-tk python3-pip
$ pip3 install --user pipenv
```

### Project dependencies

```bash
$ pipenv install
```
